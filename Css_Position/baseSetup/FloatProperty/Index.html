Understanding CSS Positioning

Floating Left removes an element from normal document flow. Then it takes that element and pushes it to the far Left as possible. Other elemetns move up as far as possible to flow around the element and take its original space.

Floating Right removes an element from normal document flow. Then it takes that element and pushes it to the far Right as possible. Other elemetns move up as far as possible to flow around the element and take its original space.
